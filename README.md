# Crypto Pro Golang Binding

Golang binding for certificated russian GOST Crypto Provider [CryptoPro CSP](https://cryptopro.ru/products/cryptopro-csp)

## Now supported 

* Open Close Store
* List ceritificates
* Find Certificate by SHA1 signature
* Low Level Sign
* Low Level Verify
* Low Level Encrypt
* Low Level Decrypt
* Generate Hash
* Sign Hash
* Get CRL's
